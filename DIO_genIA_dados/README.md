# 🧬 Bootcamp Inteligência Artificial Generativa e Dados — DIO

Este módulo centraliza os projetos práticos, desafios e estudos focados em IA Generativa, processamento de dados e engenharia de prompts desenvolvidos durante o bootcamp da DIO.

## Projetos e Conteúdos

* **[Projeto Assistente de Voz](./Projeto_assistenteDeVoz/)**: Desenvolvimento de um assistente de voz integrado com Whisper e APIs de IA.
* **[Treinando NotebookLM](./Treinando_NotebookLM/)**: Estudos de Inteligência Artificial voltados para Educação Financeira Básica utilizando a ferramenta NotebookLM do Google.
* **[E-Cards SQL e IA](./e_cards_SQL_IA/)**: Integração de scripts SQL e geração automatizada de dados utilizando prompts analíticos.
* **[Excel ETL](./Excel_ETL/)**: Laboratórios e manipulação de dados utilizando processos de ETL com Power Query.
